Hello World

For a hello world program in C++, state you are using the standard namespace 'std,' include the default input and output stream, declare the main function, and output "Hello World!" 

  using namespace std;
  #include <iostream>
  int main() {
     cout << "Hello World!";
  }
Variable Names

Each variable needs a name that identifies it and distinguishes it from the others. A valid identifier is a sequence of one or more letters, digits, or underscore characters (_). Spaces, punctuation marks, and symbols cannot be part of an identifier. Identifiers cannot be any of the reserved keywords. In addition, identifiers must always begin with a letter or an underline character. In no case can they begin with a digit.

Declaring Variables

To declare a variable in C++, one must state the data type and then follow that with a valid variable name. An example would be: 

  int a;

You can also declare multiple variables at once if they are of the same data type, such as:

  float x, y, z;

Data types

There are several data types in C++. They broadly fall into five categories: character, signed integer, unsigned integer, floating-point, and other.

The character types include:

  • char
  • char16_t
  • char32_t
  • wchar_t

The signed integer types include

  • signed char
  • signed short int
  • signed int
  • signed long int
  • signed long long int

The unsigned integer types include

  • unsigned char
  • unsigned short int
  • unsigned int
  • unsigned long int
  • unsigned long long int

The floating-point types include:

  • float
  • double
  • long double

The other types include:

  • bool
  • void
  • nullptr
if... else

If/else statements are formulated in the syntax 

 if (condition) {
    statement
 } else if (condition) {
    statement
 } else {
    statement
 }

For example,

  if(a == 1) {
    b = 3;
  } else if (a == 2) {
    b = 5;
  } else {
    b = 7;
  }
for

For-loops initalize a variable, repeat a statement while a condition is true, and increment a variable. 

  for(int i = 0; i < 5; i++) {
      cout << i << endl;
  }
while

The while-loop repeatedly executes a statement until a given condition is met. The do-while loop behaves similarly except it evaluates the condition at the end and thus always executes at least once. Examples: 

  while (a < 3) {
    cout << a;
    a++;
  }
 
  do {
    cout << a;
  } while (a < 3);
Functions

To define a function, one must declare the data type the function returns, give the function a name to be called by, and optionally state parameters to be passed into the function. 

  int minus(int a, int b) {
    int r = a - b;
    return r;
  }
